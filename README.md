#  Foodie Frontend - Food Ordering Website

A complete **Full Stack Food Ordering Web Application** developed for a single restaurant.  
This project allows users to explore food items, add them to cart, and place orders with an interactive UI.
---
## Project Overview
This is a **full-stack web application** that includes frontend, backend, and database integration.  
It simulates a real-world food ordering system.
---
##  Features
###  Home Page
- Navigation Bar (currently non-responsive)
- Hero Section with "Checkout Menu" button
- Authentication Section:
  - Sign Up
  - Login
  - Admin Login
- "What's on your mind?" section:
  - Cake
  - Juice
  - Shake
  - Pizza
  - Veg / Non-Veg
  - Breakfast / Lunch / Dinner  
- Category navigation (cake.html, pizza.html, etc.)
- Footer
---

---
##  How to Run the Project
1. Clone the repository  
git clone https://github.com/RishitaMishra1415/Foodie-frontend.git  
2. Open project folder  
3. Go to server folder  
cd server  
4. Run backend  
node server.js  
5. Open index.html in browser
